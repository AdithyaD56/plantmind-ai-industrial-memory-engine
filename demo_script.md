# Demo Script

## Opening

`PlantMind AI is an industrial memory system that learns from every report, inspection, and failure.`

## Flow

1. Upload a maintenance report for `Pump P101`
2. Upload an incident report showing vibration-driven downtime
3. Upload a safety SOP
4. Show the dashboard alert:
   - Potential Failure Detected
   - Similar incidents
   - Recommended inspection within 48 hours
5. Ask the copilot:
   - `Why is Pump P101 at risk?`
6. Show citations from the source documents

## Judge Narrative

- We do not retrieve documents like a chatbot
- We learn industrial memory patterns over time
- We connect equipment, incidents, failure modes, and procedures
- We proactively warn before the next failure
